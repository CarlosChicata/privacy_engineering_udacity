About this data:

Udacity Health Sample Data is a curated mixture of private health information (PHI) and personally identifiable information (PII) both artificially generated and extracted from Free and Open Source health data initatives which includes:

GenerateData - GPLv3
http://benkeen.github.io/generatedata/

Common Oncology Data Elements eXtensions (Codex) mCODE Test Data
https://confluence.hl7.org/display/COD/mCODE+Test+Data

Synthea and the Standard Health Record Collaborative
https://synthetichealth.github.io/synthea

Fake Apache Log Generator 
https://github.com/kiritbasu/Fake-Apache-Log-Generator
Under Apache2.0 License
 

The identities and personal information of individuals in these data sets, although seemingly realistic, is 100% fake.
Any similarities or likenesses to actual people is strictly coincidental.
